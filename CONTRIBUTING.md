# Contributing

Thank you for considering contributing! To keep contributions consistent, this repository uses a Pull Request template.

Please follow the template located at [.github/PULL_REQUEST_TEMPLATE.md](.github/PULL_REQUEST_TEMPLATE.md) when opening PRs. The template contains the required summary, verification results, and compliance checklist.

Small changes: open a PR with a concise description and link to any related issue.

Feature or breaking changes: include tests or verification steps and update documentation as needed.

Thanks!
